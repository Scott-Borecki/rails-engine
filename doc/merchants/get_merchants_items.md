# Get a Merchant's Items

```
GET /merchants/{merchant_id}/items
```


## Parameters

Name                | Type    | In    | Description
--------------------|---------|-------|--------------
`merchant_id`       | integer | path  |


## Code Sample

From your local development web server:

```
http://localhost:3000/api/v1/merchants/1/items
```
